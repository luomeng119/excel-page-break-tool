## V1 版本需求（已完成）

### 1.1 功能概述
基础版Excel分页符处理工具，用于在指定行上方添加分页符。

### 1.2 核心功能
| 功能模块 | 描述 |
|---------|------|
| 文件处理 | 读取.xlsx格式Excel文件 |
| 分页规则 | 在第一列值为"001"的行上方添加分页符 |
| 视觉标记 | 将"001"行背景色标记为红色 |
| 版本管理 | 输出文件自动添加日期和版本号（如：文件名_20260417_V1.xlsx） |

### 1.3 技术实现
- 使用Python + openpyxl库
- 识别数字格式为"000"且值为1的单元格
- 支持命令行和批处理脚本运行

### 1.4 文件输出
- 原始文件备份：文件名_原始备份.xlsx
- 处理后文件：文件名_YYYYMMDD_V1.xlsx

## V2 版本需求（待开发）

### 2.1 产品形态
桌面应用程序（.exe可执行文件），带图形用户界面(GUI)。

### 2.2 核心功能需求

#### 2.2.1 文件输入方式
| 输入方式 | 说明 |
|---------|------|
| 单文件模式 | 用户选择单个.xlsx文件进行处理 |
| 批量目录模式 | 用户选择文件夹，自动处理目录下所有.xlsx文件 |

#### 2.2.2 智能序号识别规则
支持多种位数序号自动识别和统一处理：

| 原始格式 | 识别规则 | 处理方式 |
|---------|---------|---------|
| 2位数字（如：01） | 值=1，格式="00" | 自动补零 → 001 |
| 3位数字（如：001） | 值=1，格式="000" | 保持 → 001 |
| 4位数字（如：0001） | 值=1，格式="0000" | 自动补零 → 0001 |

**统一规则**：检测到任意位数的"1"开头序号（01/001/0001等），均视为分页点。

#### 2.2.4 打印预览校验（核心功能）
- **功能**：模拟真实打印效果，在窗口中预览分页结果
- **显示内容**：
  - 分页线位置（虚线标注）
  - 每页起始行内容预览
  - 红色标记的序号行高亮显示
- **交互**：支持缩放、翻页查看

#### 2.2.5 输出规范
- 输出文件名格式：`原文件名_YYYYMMDD_V2.xlsx`
- 自动备份原始文件
- 处理日志记录

### 2.5 异常处理

| 异常场景 | 处理方式 |
|---------|---------|
| 文件被占用 | 提示关闭文件后重试 |
| 非Excel文件 | 跳过并记录日志 |
| 无序号行 | 提示未检测到分页标记 |
| 权限不足 | 提示以管理员身份运行 |
| 内存不足 | 分批处理大文件 |
